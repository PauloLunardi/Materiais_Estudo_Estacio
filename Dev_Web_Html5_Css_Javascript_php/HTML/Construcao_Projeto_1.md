# Projeto 1: Inserindo um formulário em uma página web

## Elementos Práticos

Criar a estrutura básica de uma página HTML e o container do formulário, contendo campos para inserção de :
  - nome
  - E-mail
  - Telefone de contato
  - calendário
  - data de nascimento
  - inserção de texto livre
  - test final no navegador

## Motivação
- As paginas HTML são muito populares e intuitivas.
- Velocidade de desenvolvimento.
- Acelerar a capacidade de obter informações
